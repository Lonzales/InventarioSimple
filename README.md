# InventarioSimple
Esta es una prueba de inventario utilizando PHP para la comunicación con la base de datos.
___
# Herramientas utilizadas
- Vs Code
- XAMPP
- MySQL
___
# Pasos de instalación
1. Descargar e instalar [XAMPP](https://www.apachefriends.org/es/index.html) instalando unicamente Apache y MySQL recordado la ubicación de su instalación.
2. Buscar en consola la carpeta de instalación de XAMPP y ubicar la carpeta htdocs.
3. Dentro de la carpeta htdocs ejecutar el siguiente código en la consola:
```
git clone https://github.com/Lonzales/InventarioSimple.git
```
4. Ejecutar XAMPP como administrador iniciando el servido Apache, y buscar en el navegador localhost.
